Chapter 14. OLED와 RTC를 이용한 스마트 시계

고해상도 0.96인치 OLED와 정밀 시간 모듈(RTC)을 이용해 나만의 탁상시계를 만듭니다. I2C 버스 하나에 두 개의 장치를 연결하는 방법(Daisy Chain)을 학습합니다.

I2C 병렬 연결: OLED와 RTC의 SDA/SCL을 Pico의 GP8/GP9에 공통으로 연결합니다.

14.1 스마트 시계 소스 코드

사전 준비: Thonny 패키지 관리자에서 ssd1306 라이브러리를 설치해야 합니다.

import machine
import ssd1306
import time

# 1. I2C 설정 (SDA=GP8, SCL=GP9)
i2c = machine.I2C(0, sda=machine.Pin(8), scl=machine.Pin(9))

# 2. OLED 설정 (128x64 해상도)
oled = ssd1306.SSD1306_I2C(128, 64, i2c)

# 3. Pico 내장 RTC (또는 외부 RTC 모듈) 시간 설정
# (연, 월, 일, 요일, 시, 분, 초, 마이크로초)
rtc = machine.RTC()
rtc.datetime((2024, 1, 1, 0, 12, 0, 0, 0)) # 초기 시간 세팅

print("Smart Clock 가동...")

while True:
    # 현재 시간 가져오기
    t = rtc.datetime()
    
    oled.fill(0) # 화면 지우기
    
    # 날짜 출력
    oled.text(f"{t[0]}-{t[1]:02d}-{t[2]:02d}", 20, 10)
    
    # 시간 출력 (크게 표현하기 위해 텍스트 위치 조정)
    time_str = f"{t[4]:02d}:{t[5]:02d}:{t[6]:02d}"
    oled.text("TIME:", 45, 30)
    oled.text(time_str, 35, 45)
    
    oled.show()
    time.sleep(1)

💡 I2C 주소 확인

OLED는 보통 0x3C, RTC는 0x68 주소를 가집니다. Chapter 9의 스캔 코드를 활용해 두 장치가 모두 인식되는지 먼저 확인하세요.
