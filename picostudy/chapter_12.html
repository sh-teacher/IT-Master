Chapter 12. MQTT와 Adafruit IO 클라우드 연동

IoT의 핵심 프로토콜인 MQTT를 학습합니다. 내 방의 센서 데이터를 전 세계 어디서든 스마트폰 대시보드로 모니터링할 수 있는 시스템을 구축합니다.

12.1 Adafruit IO 설정 및 라이브러리

1. Adafruit IO에 가입하여 AIO Key를 복사해 두세요.
2. Thonny의 [Tools] -> [Manage Packages]에서 umqtt.simple을 검색해 설치해야 합니다.

import time
import machine
import network
from umqtt.simple import MQTTClient

# 1. WiFi & Adafruit 설정
WIFI_SSID = "your_wifi_name"
WIFI_PASS = "your_wifi_password"
AIO_USER = "your_username"
AIO_KEY = "your_aio_key"
AIO_FEED = AIO_USER + "/feeds/temperature"

# 2. WiFi 접속 함수 (Ch 4 참고)
wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(WIFI_SSID, WIFI_PASS)
while not wlan.isconnected(): time.sleep(1)
print("WiFi Connected!")

# 3. MQTT 연결 및 전송
client = MQTTClient("Pico2_Client", "io.adafruit.com", user=AIO_USER, password=AIO_KEY)

try:
    client.connect()
    print("Connected to Adafruit IO!")
    
    while True:
        # 가상의 센서 데이터 (나중에 DHT11과 결합 가능)
        temp_data = 25.5 
        
        print(f"Publishing: {temp_data}")
        client.publish(AIO_FEED, str(temp_data))
        
        time.sleep(10) # Adafruit 무료 티어는 요청 간격을 두는 것이 좋습니다.
        
except Exception as e:
    print("Error:", e)
finally:
    client.disconnect()
