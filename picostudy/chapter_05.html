Chapter 5: 실전 IoT - 웹 서버 구축을 통한 원격 LED 제어

5.1 동작 원리 및 아키텍처

Pico 2 W 보드가 IP를 할당받은 후 TCP 80번 포트를 열어 웹 서버 역할을 수행합니다. 사용자가 스마트폰 브라우저로 접속하면 보드가 HTML 버튼 양식을 띄워주고 클릭 시 LED를 제어합니다.

[Image of IoT Client-Server Architecture for Pico 2 W]

5.2 통합 사물인터넷(IoT) 소스 코드

import network
import socket
import time
from machine import Pin

# 네트워크 연결
ssid = '와이파이_이름'
password = '비밀번호'
led = Pin("LED", Pin.OUT)

wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(ssid, password)

while not wlan.isconnected():
    time.sleep(1)

ip = wlan.ifconfig()[0]
print(f"🎉 웹 서버 가동! 브라우저 주소: http://{ip}")

# 소켓 바인딩
addr = socket.getaddrinfo('0.0.0.0', 80)[0][-1]
s = socket.socket()
s.bind(addr)
s.listen(1)

def web_page():
    return '''<!DOCTYPE html>
    <html>
    <body style="text-align:center; font-family:sans-serif; margin-top:50px;">
        <h1>Pico 2 W 원격 제어 센터</h1>
        <a href="/?led=on"><button style="padding:15px; background:lightgreen;">LED 켜기</button></a>
        <a href="/?led=off"><button style="padding:15px; background:salmon;">LED 끄기</button></a>
    </body>
    </html>'''

while True:
    cl, addr = s.accept()
    request = cl.recv(1024).decode('utf-8')
    
    if '/?led=on' in request:
        led.value(1)
    if '/?led=off' in request:
        led.value(0)
        
    response = web_page()
    cl.send('HTTP/1.0 200 OK\r\nContent-type: text/html\r\n\r\n')
    cl.send(response)
    cl.close()
