02. 변수와 자료형: 데이터 마법 상자

1. 변수(Variable)란?

변수는 데이터를 담기 위해 메모리에 이름을 붙인 '상자'입니다. 이 상자에 숫자를 넣으면 정수형 변수, 글자를 넣으면 문자형 변수가 됩니다.

▲ 변수: 메모리라는 거대한 창고에 라벨(이름)을 붙인 상자를 만드는 것

2. 🎨 창의적 예제: 용사의 스테이터스

#include <stdio.h>
int main() {
    int level = 10;          // 정수 상자 (레벨)
    float attack = 12.5;     // 실수 상자 (공격력)
    char rank = 'S';         // 문자 상자 (등급)

    printf("용사의 등급: %c\n", rank);
    printf("현재 레벨: %d\n", level);
    printf("현재 공격력: %.1f\n", attack);
    return 0;
}
