07. 배열 (Array): 데이터 기차

1. 배열이란 무엇인가요?

변수가 상자 하나라면, 배열은 같은 크기의 상자들이 기차처럼 쭉 이어져 있는 형태입니다. 중요한 점은 상자 번호(인덱스)가 0번부터 시작한다는 것이죠!

▲ 배열: 여러 데이터를 한 번에 관리하는 효율적인 바구니

2. 🎒 창의적 예제: RPG 아이템 인벤토리

#include <stdio.h>
int main() {
    // 5개의 아이템 수량을 담는 배열
    int inventory[5] = {10, 20, 0, 5, 1};

    printf("--- 나의 인벤토리 ---\n");
    for(int i = 0; i < 5; i++) {
        printf("%d번 칸 아이템 수량: %d개\n", i, inventory[i]);
    }
    return 0;
}
