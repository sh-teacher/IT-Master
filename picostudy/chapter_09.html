Chapter 9. I2C LCD 디스플레이 문자열 출력

컴퓨터 모니터 없이도 보드의 현재 상태나 센서 데이터 값을 시각적으로 뿌려줄 수 있는 I2C 규격의 LCD1602 모듈을 통제해 봅니다.

9.1 I2C 디바이스 주소 스캔 및 라이브러리 준비

I2C LCD를 다루려면 외부 구동용 외부 라이브러리(pico_i2c_lcd.py) 파일이 내장 공간에 미리 업로드되어 있어야 합니다. 우선 하드웨어가 올바른 물리 채널에 접속되었는지 확인하는 탐색 코드입니다.

from machine import I2C, Pin

# Pico 2의 I2C0번 채널 활성화 (SDA: GPIO 8, SCL: GPIO 9)
i2c = I2C(0, sda=Pin(8), scl=Pin(9), freq=400000)

print("I2C 무선 장치 스캔 시작...")
devices = i2c.scan()

if len(devices) == 0:
    print("연결된 I2C 부품을 찾지 못했습니다. 배선을 확인하세요.")
else:
    for device in devices:
        print(f"발견된 디바이스 16진수 고유 주소: {hex(device)}")

9.2 LCD 화면에 센서 실시간 문자열 시각화

LCD 화면에 고정 문자를 찍고, 아래 행에는 실시간 데이터 변동 수치를 출력하는 정형화 대시보드 출력 코드입니다.

from machine import I2C, Pin
import time
# 미리 Thonny 패키지로 저장해둔 LCD 모듈 컴포넌트 임포트
from pico_i2c_lcd import I2cLcd

i2c = I2C(0, sda=Pin(8), scl=Pin(9), freq=400000)
# 위 스캔에서 발견된 16진수 주소값(보통 0x27) 입력
lcd = I2cLcd(i2c, 0x27, 2, 16)

# 화면 초기화 및 백라이트 점등
lcd.clear()
lcd.backlight_on()

# 1행에 텍스트 고정 출력 (0번방 열, 0번방 행)
lcd.move_to(0, 0)
lcd.putstr("Pico 2 IoT Monitor")

count = 0
while True:
    # 2행 0열로 커서 이동 후 실시간 데이터 교체
    lcd.move_to(0, 1)
    lcd.putstr(f"System Running:{count}")
    
    count += 1
    time.sleep(1.0)
