Chapter 3: 기초 제어 - 내장 LED 깜빡이기 (Blink)

3.1 하드웨어 제어를 위한 약속

Pico 2 W 모델은 무선 칩이 내장 LED를 제어하기 때문에 핀 번호를 숫자 25가 아닌 문자열 "LED"로 명시해야 합니다.

3.2 예제 코드

아래 코드를 작성한 후 보드 내부에 main.py로 저장하세요.

from machine import Pin
import time

# 숫자 25 대신 문자열 "LED"를 사용하여 객체 생성
led = Pin("LED", Pin.OUT)

# 무한 루프를 돌며 0.5초 간격으로 토글
while True:
    led.toggle()
    time.sleep(0.5)

참고: 파일명을 main.py로 저장하면, PC와 분리 후 배터리만 연결해도 전원이 켜지자마자 스스로 위 코드를 실행합니다.
