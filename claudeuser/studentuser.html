Claude 계정 조회

학교명 · 학년-반 · 이름을 입력하면 배정된 Claude 계정을 확인할 수 있어요

학년과 반을 붙임표(-)로 구분해서 입력하세요. 예) 2학년 3반 → 2-3
입력한 정보:
Claude 계정 (이메일)
비밀번호
구독 기간
본인의 계정 정보이니 다른 사람과 공유하지 마세요.

로그인 방법 안내

아래 순서대로 진행하면 바로 로그인할 수 있어요.

Claude 로그인 페이지 바로가기 →
1
위 버튼을 눌러 Claude 로그인 페이지로 이동합니다.
2
로그인 화면에서 “Google로 계속하기(Continue with Google)” 버튼을 클릭합니다.
Claude 로그인 화면 예시
Google로 계속하기
※ 실제 화면 디자인은 업데이트에 따라 조금 다를 수 있어요.
3
위에서 확인한 계정(이메일)과 비밀번호를 Google 로그인 화면에 입력합니다.
4
로그인이 완료되면 Claude를 바로 사용할 수 있어요.
5
로그인 후에는 반드시 비밀번호를 변경하세요. 처음 지급된 비밀번호는 다른 사람도 알고 있을 수 있어 그대로 사용하면 안전하지 않아요.
비밀번호 변경하러 가기 →
Claude 로그인 후 > 설정(Settings) > 계정(Account)에서도 변경할 수 있어요.
