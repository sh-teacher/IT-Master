Chapter 4. 무선 Wi-Fi 연결 테스트

이제 Pico 2 W의 핵심인 무선 인터넷 기능을 활성화해 보겠습니다.

4.1 네트워크 주파수 확인

⚠️ 중요한 제약 조건

Pico 2 W는 2.4GHz 대역의 와이파이만 지원합니다. 공유기 이름 끝에 '5G'가 붙은 주파수에는 연결할 수 없으므로 주의하세요.

2.4GHz 무선 네트워크를 통한 인터넷 세상으로의 연결

4.2 연결 코드 실습

import network
import time

ssid = '여기에_와이파이_이름'
password = '여기에_비밀번호'

def connect():
    wlan = network.WLAN(network.STA_IF) # 스테이션 모드 활성
    wlan.active(True)
    wlan.connect(ssid, password)
    
    print("연결 중...", end="")
    while not wlan.isconnected():
        print(".", end="")
        time.sleep(1)
        
    print("\n🎉 연결 성공!")
    print("보드 IP 주소:", wlan.ifconfig()[0])

connect()

4.3 결과 확인

코드를 실행했을 때 하단 셸 창에 192.168.x.x와 같은 숫자가 나오면 성공입니다. 이 숫자는 보드가 공유기로부터 부여받은 고유 주소(IP)입니다.
